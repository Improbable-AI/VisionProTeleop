Three Pillars of the System

visionOS

Tracking Streamer

The core visionOS app running on Apple Vision Pro. Captures hand, wrist, and head tracking data with sub-frame precision and streams it to connected clients.

  • Real-time hand skeleton tracking
  • Global SLAM pose estimation
  • WebRTC video/audio reception
  • AR simulation rendering
  • UVC camera support

Python

avp_stream

Python client library that receives tracking data and sends video, audio, or MuJoCo simulations back to Vision Pro. Install with pip and start building.

  • Simple streaming API
  • Frame callbacks for processing
  • Stereo video support
  • MuJoCo integration
  • Coordinate frame transforms
View on PyPI
Get Started

Up and Running in Minutes

Python 
# Install the package
pip install --upgrade avp_stream

# Basic usage
from avp_stream import VisionProStreamer

avp_ip = "10.31.181.201"  # Your Vision Pro IP
s = VisionProStreamer(ip=avp_ip)

while True:
    r = s.latest
    print(r['head'], r['right_wrist'], r['right_fingers'])
